# Red Team Infrastructure Deployment

This repository contains Terraform and Ansible configurations to automatically deploy a complete red team infrastructure in AWS, including:
- Sliver C2 Server
- Redirector
- Attacker Workstation

## Prerequisites

- AWS Account
- Terraform
- Ansible
- Python3
- AWS CLI
